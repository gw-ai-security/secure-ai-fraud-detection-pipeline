# Data Minimization & Privacy-by-Design

## Objective
This document explains how **data minimization** and **privacy-by-design** principles are applied throughout the project.

---

## Synthetic Data Only
All datasets in this repository are artificially generated and safe for public sharing.

---

## Excluded Data
- User IDs
- Account numbers
- Customer identifiers
- Merchant identifiers

These are never used as model features.

---

## Technical Enforcement
Sensitive columns are excluded before encoding and scaling.

---

## Disclaimer
This project demonstrates engineering practices only and does not claim legal compliance.
